Overview

Mcaster1BackDraft is a pure C++ WAF and nginx log analyzer. Single binary, three ports, one process. It sits behind nginx, intercepts proxied traffic on a per-page basis, inspects requests against a rule engine, scores threats, logs everything to MySQL, and forwards clean requests to PHP-FPM via FastCGI.

WAF Engine

9 built-in rules covering SQLi, XSS, path traversal, command injection, PHP exploits, empty user agents, and non-standard HTTP methods. URL-decoded inspection catches encoded attacks. Per-request threat scoring with configurable thresholds.

Log Analysis

Chart.js-powered dashboards parsing nginx access and error logs directly. Hourly traffic breakdown, status code distribution, top pages, top IPs, top user agents, top referers. Per-site drill-down reports with 6 tabs.

Multi-Site

Discovers all nginx sites from /etc/nginx/sites-enabled/. Per-site WAF enrollment with per-page location block generation. Currently 27 sites discovered on this server.

Single Binary

One C++ daemon, three ports (9432 WAF, 8862 UI, 8832 API), bounded thread pools auto-sized from system CPU and memory. Built with autotools. No external runtime dependencies beyond MySQL and PHP-FPM.
