Getting Started

Build, configure, and run Mcaster1BackDraft

HomeArchitectureGetting StartedWAF RulesRoadmap
1

Install Dependencies

sudo apt install build-essential autoconf automake \
  libssl-dev libyaml-cpp-dev libmariadb-dev nlohmann-json3-dev
2

Build

cd /var/www/mcaster1.com/Mcaster1BackDraft
./autogen.sh
./configure
make -j$(nproc)

Binary at: src/Mcaster1BackDraft

3

Create Database

mysql --defaults-extra-file=~/.my.cnf < sql/schema.sql

Creates mcaster1_backdraft with 16 tables, seeds 9 WAF rules, 14 agent signatures, and the admin user.

4

Configure

Edit etc/Mcaster1BackDraft.yaml:

  • Set database credentials
  • Set TLS cert/key paths
  • Set waf.upstream_fcgi to your PHP-FPM socket
  • Set auth.master_username / auth.master_password
5

Run

./src/Mcaster1BackDraft -c etc/Mcaster1BackDraft.yaml

Three ports start: 9432 (WAF), 8862 (Web UI), 8832 (API).

6

Login

Browse to https://your-server:8862/login.php and sign in with your master credentials.

7

Enable WAF on a Page

Add a location block in your nginx site config:

location = /page.php {
    proxy_pass https://127.0.0.1:9432;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-BackDraft-Page "site.com/page.php";
    proxy_buffering off;
}
sudo nginx -t && sudo nginx -s reload

Traffic for that page now flows through BackDraft. Check the dashboard for logged requests.

Verify

# Health check
curl -sk https://127.0.0.1:9432/health

# API status
curl -sk -H "Authorization: Bearer YOUR_TOKEN" https://127.0.0.1:8832/api/status

# Test SQLi detection
curl -sk -H "Host: your-site.com" \
  "https://127.0.0.1:9432/test.php?id=1%20UNION%20SELECT%20*"

# Check DB
mysql --defaults-extra-file=~/.my.cnf mcaster1_backdraft \
  -e "SELECT path, threat_score FROM backdraft_requests ORDER BY id DESC LIMIT 5"
